# Ranoviq Technologies

Official website for **Ranoviq Technologies** — a Udyam registered Micro Enterprise (UDYAM-MH-20-0325194) based in Nagpur, Maharashtra. We offer computer programming, IT consultancy, website development, digital marketing, and managed IT solutions.

## Live Site

[https://www.ranoviq.com](https://www.ranoviq.com)

## Tech Stack

- **HTML5** — Semantic markup, accessibility-friendly
- **CSS3** — Custom styling, CSS variables, responsive layout, animations
- **JavaScript** — Navigation toggle, back-to-top, scroll reveal, form handling
- **Font Awesome** — Icons
- **Google Fonts** — Poppins

No build step or framework; static files only.

## Project Structure

```
ranoviq/
├── index.html              # Homepage (Hero, Services, About, Process, Why Us, Portfolio, Testimonials, Contact)
├── faq.html                # Frequently Asked Questions
├── privacy-policy.html     # Privacy Policy
├── terms-and-conditions.html # Terms and Conditions
├── css/
│   └── style.css           # Main stylesheet (brand colors, layout, components)
├── js/
│   └── script.js          # Navigation, back-to-top, scroll effects, contact form
├── assets/
│   └── images/             # Logos, illustrations, project images
├── .gitignore
└── README.md
```

## Features

- **Responsive design** — Mobile, tablet, and desktop
- **Single-page homepage** — Smooth scroll navigation to sections
- **Separate legal/help pages** — FAQ, Privacy Policy, Terms and Conditions
- **Footer useful links** — Quick access to FAQ and policies
- **Card hover effects** — With touch-friendly `:active` feedback on mobile
- **Back-to-top button** — Sticky on scroll
- **SEO & social** — Meta tags, Open Graph, Twitter Card, canonical URL

## How to Run Locally

1. Clone the repository:
   ```bash
   git clone https://github.com/pranali-wapikon/ranoviq.git
   cd ranoviq
   ```

2. Open in browser:
   - Double-click `index.html`, or
   - Use a local server (e.g. Live Server in VS Code, or `npx serve .`)

No `npm install` or build is required.
